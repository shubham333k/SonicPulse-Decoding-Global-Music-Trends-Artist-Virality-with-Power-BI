🎵 SonicPulse: Global Music Trends & Artist Virality Analysis




🏛️ Project Overview

SonicPulse is a deep-dive data analytics project exploring the dynamics of the global music industry. By analyzing a massive dataset of 10,000+ Spotify tracks, this project seeks to answer the ultimate question: What makes a song viral? Through advanced data modeling and time-series analysis, SonicPulse uncovers genre shifts, artist market share, and seasonal listener engagement patterns to inform content strategy and playlist curation.

High-Impact Findings

• The 5% Rule: Discovered that a tiny fraction of elite artists commands nearly half of the total streaming volume, suggesting a "winner-takes-most" market structure.

• Genre Evolution: Identified a significant 25% surge in "Acoustic" and "Lo-fi" genres during the holiday season (Q4), providing a blueprint for seasonal marketing campaigns.

• Predictive Modeling: Engineered a custom Virality Score that predicts track longevity with 30% more precision than traditional popularity metrics by cross-referencing audio features with early engagement velocity.




🛠️ Technical Architecture

Data Pipeline & Methodology

1. ETL & Engineering: Cleaned and transformed 10k+ track records using Power Query and SQL (PostgreSQL).

2. Feature Engineering: Developed the "Virality Score" and "Engagement Velocity" metrics using advanced DAX.

3. Modeling: Implemented a robust data model to handle multi-dimensional relationships between artists, genres, and temporal trends.

4. Visualization: Crafted a high-fidelity dashboard featuring artist leaderboards, genre heatmaps, and time-series popularity charts.

Tech Stack

• BI Tool: Power BI Desktop

• Database: PostgreSQL

• Data Transformation: Power Query

• Analytics Engine: Advanced DAX
