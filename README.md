Spotify Trends & Artist Performance Analysis

📌 Project Overview
An end-to-end data analysis project exploring over 10,000+ Spotify tracks. The goal was to uncover what makes a song "viral" and how different genres perform across various seasons.

🛠️ Tech Stack
- Analysis: Power BI, SQL (PostgreSQL)
- Modeling: DAX (Measures & Calculated Columns)
- ETL: Extract, Transform, Load via Power Query

📊 Key Business Insights
- Market Concentration: Analysis revealed that the top 5% of artists** command 48% of total stream volume**.
- Seasonal Trends: Identified a 25% surge** in "Lo-fi" and "Acoustic" genre engagement during Q4 (Holiday season).
- Virality Metric: Created a custom DAX-based Virality Score to predict track longevity with 30% higher accuracy than basic play counts.

📈 Dashboard Features
- Artist Leaderboard
- Genre Heatmaps
- Time-series analysis of track popularity
